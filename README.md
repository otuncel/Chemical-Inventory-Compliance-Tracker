# Chemical Inventory Compliance Tracker

An automation tool designed for Chemical Engineers to manage inventory and ensure regulatory compliance.

## Key Features
- **Live Excel Integration:** Real-time updates using `pywin32` without file access errors.
- **PubChem API:** Automatically fetches IUPAC names and compound details using CAS numbers.
- **Compliance Check:** Flags prohibited or restricted substances based on a local regulatory database.
- **VBA Powered:** One-click execution directly from the Excel interface.
